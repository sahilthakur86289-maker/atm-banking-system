# ATM Banking System

This is a simple Python mini project based on an ATM Banking System. It uses Object Oriented Programming concepts.

## Features

- Deposit money
- Withdraw money
- Check account balance
- Show transaction history
- User can set PIN
- PIN based login

## Concepts Used

- Classes and objects
- Methods
- If-else conditions
- While loop
- List
- User input

## How to Run

1. Install Python on your computer.
2. Open this project folder.
3. Run this command:

```bash
python main.py
```

## Login

When the program starts, the user can enter account holder name, set a PIN, and enter starting balance.
After account creation, the same PIN is used for login.
